# Research: Participatory Chanting Proof of Concept (PoC)

## Introduction
The goal of participatory chanting is to allow a user to chant along with a recording, or practice interactive call-and-response chanting.

## Proposed Approaches
1. **Call and Response Mode**:
   - The application plays a line (guru part).
   - The application pauses or listens, prompting the user to chant it back (shishya part).
   - Audio feedback or pronunciation analysis (using Speech-to-Text or phonetic matching models) can be integrated.

2. **Parallel Chanting Mode**:
   - Real-time vocal alignment using pitch detection/audio correlation.
