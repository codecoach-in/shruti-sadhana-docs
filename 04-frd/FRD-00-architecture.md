# FRD-00: System Architecture

## Functional Requirements
- The system must support real-time audio playback synchronization with active text.
- The UI must handle Sanskrit typography, including conjunct consonants and accent marks.
- Support offline capabilities (audio caching, offline-first reading).

## Technical Requirements
- Web Technologies: HTML5, CSS3, JavaScript.
- Audio Control: Web Audio API or custom HTML5 audio wrappers.
