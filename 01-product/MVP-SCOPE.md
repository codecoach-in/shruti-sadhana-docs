# MVP Scope

## Purpose

This document defines what is included and excluded from the first public release of Shruti Sadhana.

The goal of the MVP is to validate demand for structured Sanskrit Stotram learning and establish a high-quality learning experience before investing in advanced AI capabilities.

---

# MVP Objectives

The MVP should allow a devotee to:

1. Discover available stotram courses
2. Learn verses in a structured manner
3. Listen to authentic guided recitation
4. Practice consistently
5. Track learning progress
6. Experience a devotional digital environment

---

# Included Features

## Authentication

- Sign Up
- Login
- Password Reset
- Profile Management

---

- Welcome Screens

- Initial Personalization

- Telugu

## Home

## Learning Library

- Browse Courses

## Lesson Experience

Each lesson includes:

- Sanskrit Text
- Transliteration
- Meaning

Audio Modes:

- Normal Speed (Free)
- Slow Speed (Premium)
- Medium Speed (Premium)

## Puja Ghar

# Content Included at Launch

Initial launch should contain at least one flagship course.

Priority candidates:

1. Lalita Sahasranama
2. Vishnu Sahasranama
3. Hanuman Chalisa

# Explicitly Excluded from MVP

## AI Features

- Pronunciation feedback
- Pronunciation scoring
- Speech recognition
- Personalized corrections

## Community Features

- Discussion forums
- User groups
- Social sharing
- Messaging

## Live Learning

- Live classes
- Live workshops
- Teacher interactions

## Gamification

- Badges
- Streak rewards
- Leaderboards

## Advanced Analytics

- Detailed pronunciation analytics
- Learning recommendations
