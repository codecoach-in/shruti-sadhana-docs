# MVP Scope

## Purpose

This document defines what is included and excluded from the first public release of Shruti Sadhana.

The goal of the MVP is to validate demand for structured Sanskrit Stotram learning and establish a high-quality learning experience before investing in advanced AI capabilities.

# Included Features

## Authentication
- Sign Up
- Login
- Password Reset
- Profile Management

## Onboarding
- Welcome Screens
- Script Preference Selection
- Initial Personalization

## Home
- Continue Learning
- Recent Lessons
- Featured Courses
- Enter Puja Ghar

## Learning Library
- Browse Courses
- Course Detail Page
- Lesson Index
- Teacher Information
- Progress Overview

## Lesson Experience
- Sanskrit Text
- Transliteration
- Meaning
- Audio Playback
- Normal Speed (Free)
- Slow Speed (Premium)
- Medium Speed (Premium)

## Practice
- Repeat Playback
- Guided Practice
- Lesson Revisit

## Progress Tracking
- Lessons Completed
- Course Progress
- Recently Viewed Lessons

## Puja Ghar
- Deity Selection
- Personalized Deity Space
- Devotional Ambience
- Quick Access to Learning

## Participatory Chanting
- Guided call-and-response chanting experience
- User chanting participation windows
- Practice-oriented devotional experience

## Referrals
- Referral code generation
- Referral tracking
- Community growth support

## Donations

- Support Shruti Sadhana initiatives
- Contribution acknowledgement

## Subscription System
- Slow-speed recitation
- Medium-speed recitation
- Premium content
